# 10SE6A-Assessment-Task-1

## Installing & Running

Create a fork of the repository, and then open it in Github Desktop. Click the button which says "Open the repository in your external editor". This will be whatever you have installed.

Once opened, open the terminal and run this command:

`pip install -r requirements.txt`

After that, open `main.py` and run. The program will run in the terminal.
